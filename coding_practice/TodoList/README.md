## 🎨 CSS Design Guide

- **Font Family**: Roboto
- **Text Color**: White
- **Button Color**: Orange (`background-color: orange`)
- **Text Alignments**: Centered horizontally
- **Typography**:
  - Heading: Bold, Large
  - Subheadings & Paragraphs: Normal weight, moderate size
- **Button Styling**: Rounded edges, centered, bold white text
