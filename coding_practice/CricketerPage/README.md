# 🏏 Cricketer Page - Assignment

## 🎯 Objective

Build a **Cricketer Profile Page** using HTML and CSS (and optionally Bootstrap) that resembles the given design. You're encouraged to recreate this for your **favourite cricketer** using the same layout and styles.

---

## 📷 Reference Design

![KL Rahul Page](./cricketer-page-preview.png)

> Try to achieve the same layout, colors, fonts, and spacing as closely as possible.

---

## ✅ Requirements

### 🔹 Page Sections

1. **Top Section (Header)**:

   * Cricketer’s image (KL Rahul)
   * Name: `K.L. Rahul`
   * Description text

2. **IPL Records Section**:

   * Heading: `IPL Records`
   * Record card with:

     * Image
     * Match title
     * Record description

3. **Videos Section**:

   * Heading: `Videos`
   * 3 thumbnails with captions

4. **Button**:

   * Label: `View more`
